#  A Social Media App 
</div>


## **Features -**
- Feed Posts
- Explore & Trending Posts
- Add , Edit& Delete Post
- Add  Comment On Post
- User Profile Update
- Follow / Unfollow User
- Like and Comment on Post
- Authentication:
  - User Signup
  - User Login
  - User Logout


## **Built with -**

- React JS
- React Router v6
- Material Ui 
-- Redux Toolkit 
- Backend setup using [MockBee](https://mockbee.netlify.app/)
- React-toastify for Toasts
- Cloudinary for Hosting of Images
